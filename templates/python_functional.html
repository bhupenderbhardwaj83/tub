Core Concepts

First-Class Functions
WHAT IT DOES

Functions can be assigned to variables, passed as arguments, and returned from other functions.

EXAMPLE 
# Functions as values
def add(a, b):
    return a + b

ops = [add]  # Store in list
print(ops[0](2, 3))  # 5
Higher-Order Functions
WHAT IT DOES

Functions that take other functions as arguments or return functions.

EXAMPLE 
def apply(fn, x):
    return fn(x)

def square(n):
    return n * n

print(apply(square, 4))  # 16
Lambda Functions
EXAMPLE 
# Anonymous function
square = lambda x: x * x
print(square(5))  # 25

# Multiple arguments
add = lambda x, y: x + y
print(add(3, 7))  # 10

Built-in Functional Tools

map() - Transform Each Element
EXAMPLE 
nums = [1, 2, 3, 4, 5]

# Double each number
doubled = list(map(lambda n: n * 2, nums))
print(doubled)  # [2, 4, 6, 8, 10]

# With named function
def square(n):
    return n ** 2

squared = list(map(square, nums))
print(squared)  # [1, 4, 9, 16, 25]
filter() - Keep Some Items
EXAMPLE 
nums = [1, 2, 3, 4, 5, 6]

# Keep only even numbers
evens = list(filter(lambda n: n % 2 == 0, nums))
print(evens)  # [2, 4, 6]

# Keep numbers greater than 3
greater = list(filter(lambda n: n > 3, nums))
print(greater)  # [4, 5, 6]
reduce() - Fold to Single Value
EXAMPLE 
from functools import reduce

nums = [1, 2, 3, 4, 5]

# Sum all numbers
total = reduce(lambda a, b: a + b, nums)
print(total)  # 15

# Product of all numbers
product = reduce(lambda a, b: a * b, nums)
print(product)  # 120
zip() - Combine Sequences
EXAMPLE 
names = ["Alice", "Bob", "Charlie"]
ages = [25, 30, 35]

# Combine into pairs
paired = list(zip(names, ages))
print(paired)
# [('Alice', 25), ('Bob', 30), ('Charlie', 35)]

# Create dictionary
people = dict(zip(names, ages))
print(people)
# {'Alice': 25, 'Bob': 30, 'Charlie': 35}
any() / all() - Logical Checks
EXAMPLE 
nums = [1, 2, 3, 4, 5]

# Check if any number is greater than 4
print(any(n > 4 for n in nums))  # True

# Check if all numbers are positive
print(all(n > 0 for n in nums))  # True

# Check if all numbers are even
print(all(n % 2 == 0 for n in nums))  # False

functools Module

partial() - Pre-fill Arguments
EXAMPLE 
from functools import partial

def power(base, exponent):
    return base ** exponent

# Create square function (exponent=2)
square = partial(power, exponent=2)
print(square(5))  # 25

# Create cube function (exponent=3)
cube = partial(power, exponent=3)
print(cube(3))  # 27

Functional-Style Comprehensions

List, Dict, Set Comprehensions
EXAMPLE 
# List comprehension
squares = [x**2 for x in range(5)]  # [0, 1, 4, 9, 16]

# Dict comprehension
square_dict = {x: x**2 for x in range(5)}

# Set comprehension
evens = {x for x in range(10) if x % 2 == 0}

# Generator expression (lazy evaluation)
gen = (x**2 for x in range(1000000))  # Memory efficient
